# WIND - Weather Information for Nation Data

## Project Overview
WIND is a Flask-based web application that provides country information along with real-time weather data, currency conversion, and tourist place recommendations using external APIs.

The goal of this project is to integrate multiple data sources into a single interactive platform.

## Features
- Country information display
- Real-time weather data
- Currency converter with exchange rates
- Famous places to visit by country
- Clean web interface using HTML, CSS, Bootstrap
- API integration

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap
- REST APIs
- JavaScript

## APIs Used
- OpenWeather API
- Exchange Rate API
- Wikipedia API
